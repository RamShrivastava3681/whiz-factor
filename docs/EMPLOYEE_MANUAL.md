## Getting Started

### Logging In
1. Open your web browser and navigate to the Whizunik Factoring portal
2. Enter your username and password
3. Click "Sign In" to access the system
4. Upon successful login, you'll be directed to the main dashboard

### Navigation
- **Sidebar**: Main navigation menu on the left side
- **Top Bar**: Search, notifications, and user profile
- **Breadcrumbs**: Shows your current location in the system
- **Notification Bell**: Click to view system alerts and messages

## Managing Entities

### Adding Buyers
1. Navigate to **Entities** → **Buyers**
2. Click **"Add Buyer"** button
3. Fill in required information:
   - Legal Name
   - Registration Number
   - Industry Sector
   - Country & Address
   - Contact Details
4. Set credit limits and risk parameters
5. Click **"Create Buyer"** to save

### Adding Suppliers  
1. Go to **Entities** → **Suppliers**
2. Click **"Add Supplier"** button
3. Complete supplier information:
   - Company Details
   - Bank Information
   - Certification Status
   - Payment Terms
4. Upload required documents
5. Save the supplier profile

### Managing Entity Details
- **View**: Click on any entity to see full details
- **Edit**: Use the edit button to modify information
- **Status**: Toggle active/inactive status as needed
- **History**: Review transaction history with each entity

## Processing Transactions

### Creating a New Transaction
1. From the dashboard, click **"Add Transaction"**
2. Select transaction type:
   - Letter of Credit
   - Bank Guarantee
   - Documentary Collection
   - Trade Finance Loan
3. Fill in transaction details:
   - **Parties**: Select buyer and supplier
   - **Amount**: Enter transaction value and currency
   - **Terms**: Set payment and delivery terms
   - **Documents**: Upload required documentation
4. Review and submit for processing

### Transaction Workflow
1. **Draft**: Initial transaction creation
2. **Review**: Internal verification process
3. **Approved**: Ready for execution
4. **Active**: Transaction in progress
5. **Completed**: Successfully finished
6. **Cancelled**: Terminated before completion

### Document Management
- **Upload**: Drag and drop or browse for files
- **Review**: Preview documents within the system  
- **Version Control**: Track document revisions
- **Download**: Export documents as needed

## Fee & Limits Management

### Configuring Fees
1. Navigate to **Fee & Limits** → **Fee Configuration**
2. Set fee structures:
   - **Processing Fees**: Base transaction fees
   - **Commission Rates**: Percentage-based charges
   - **Documentary Fees**: Document handling costs
3. Configure fee tiers by:
   - Transaction amount
   - Customer type
   - Geographic region

### Setting Limits
1. Go to **Fee & Limits** → **Limit Management**
2. Establish limits for:
   - **Customer Limits**: Per-customer exposure
   - **Product Limits**: By transaction type
   - **Country Limits**: Geographic concentration
   - **Currency Limits**: Foreign exchange exposure

### Fee Calculation Preview
- Use the preview feature to test fee calculations
- Verify limit compliance before transaction approval
- Generate fee summaries for customer communications

## Treasury Operations

### Disbursement Management
1. Access **Treasury** → **Disbursements**
2. View pending payment queue
3. Process disbursements:
   - Review payment details
   - Verify beneficiary information
   - Approve or reject payments
4. Track payment status and confirmations

### Reserve Management
1. Go to **Treasury** → **Reserves**
2. Monitor reserve requirements:
   - **Regulatory Reserves**: Compliance requirements
   - **Risk Reserves**: Credit risk provisions
   - **Operational Reserves**: Working capital
3. Manage reserve holds and releases

### Financial Ledger
- **Account Balances**: Real-time account positions
- **Transaction History**: Detailed financial records
- **Reconciliation**: Match transactions with bank statements
- **Reporting**: Generate financial summaries

## Monitoring & Alerts

### Dashboard Overview
The monitoring dashboard shows:
- **Transaction Health**: Overall system status
- **Risk Indicators**: Potential issues requiring attention
- **Alert Summary**: Active notifications by severity

### Alert Management
1. Navigate to **Monitoring** → **System Alerts**
2. Review active alerts:
   - **Critical**: Immediate action required
   - **Warning**: Attention needed
   - **Info**: Informational updates
3. Resolve alerts:
   - Click **"Resolve"** on individual alerts
   - Add resolution notes
   - Use bulk actions for multiple alerts

### Risk Indicators
- **Entity Risk**: Customer creditworthiness changes
- **Transaction Risk**: High-risk transaction flags
- **Operational Risk**: System performance issues
- **Market Risk**: Currency or interest rate exposures
